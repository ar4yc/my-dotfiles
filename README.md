# My hyprland WM dotfiles

### Used:
* Terminal - Kitty
* Network Manager - NM-applet
* Bluetooth Manager - Overskride
* Volume Control - Pavucontrol
* File Manager - Dolphin
* App launcher - Wofi
* Top bar - Waybar
* Wallpapers - SwayBG, Waypaper
* Color Picker - Hyprpicker
* Screenshot - Flameshot

### Installation

* #### Clone dotfiles: #
```bash
git clone https://github.com/ar4yc/hypr-dotfiles.git && cp -r hypr-dotfiles/.config "$HOME/"
```
* #### Install Programs #
```bash
yay -S pavucontrol nm-applet overskride-bin waybar cava playerctl dolphin swaync wofi kitty gtk3 gtk4 qt6ct archlinux-xdg-menu wl-clipboard wtype cliphist flameshot-git swaybg waypaper hyprpicker hyprpolkitagent
```
